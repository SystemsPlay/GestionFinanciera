Configuración de Google Sheets

Para que la aplicación funcione correctamente, debes desplegar el script de Google Apps y autorizarlo. Luego, esta aplicación web se conectará directamente a tu Google Sheet a través del ID del script.

Asegúrate de que el script apps script.txt esté correctamente configurado y publicado como aplicación web con acceso "Cualquier persona, incluso anónimo" o "Cualquier persona" si requieres que el login funcione desde el navegador. Para un login seguro, configúralo como "Solo tú" si vas a usar tu cuenta Google, o "Cualquier persona" con un sistema de autenticación custom.
